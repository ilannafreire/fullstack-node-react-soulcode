Objetivo:

Criar uma página HTML simples para uma “página pessoal”, aplicando estilização com classes, IDs, pseudoclasses e cores em inglês.

HTML Básico:

Crie um arquivo HTML que tenha um título, um cabeçalho (header), uma seção sobre você (section), uma lista de hobbies e um rodapé (footer).

Estilos com CSS:

  • Defina uma cor de fundo usando uma classe para o cabeçalho e o rodapé.
  • Aplique uma cor de texto específica para o título da página usando uma id.
  • Estilize os itens da lista de hobbies com uma classe, usando diferentes cores para o texto.
  • Pseudoclasses: Ao passar o mouse sobre os itens da lista de hobbies, mude a cor do texto.
  • Defina uma cor diferente para os links da página antes e depois de serem visitados.

Desafio adicional:

Adicione uma borda colorida ao redor do rodapé ao passar o mouse nele usando uma pseudoclasse.
